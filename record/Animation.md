## API

### animationend 事件

监听动画结束事件

```js
element.addEventListener("animationend", event => {
  // ...
});
```

### animationstart 事件

监听动画开始事件

```js
element.addEventListener("animationstart", event => {
  // ...
});
```

### animationiteration 事件

监听动画重复事件

```js
element.addEventListener("animationiteration", event => {
  // ...
});
```
